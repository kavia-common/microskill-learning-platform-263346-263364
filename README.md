# microskill-learning-platform-263346-263364

This repo includes:
- frontend_react: React UI for micro-skill LMS with feed, detail, quiz, and creator tools.
- backend: Express API for AI-based lesson generation and local media rendering (MP4 + VTT) with optional Supabase persistence.

Quick start:
- Frontend: cd frontend_react && npm install && npm start
- Backend: cd ../microskill-learning-platform-263346-263365/backend && npm install && npm start

Use REACT_APP_API_BASE to point the frontend to the backend URL (e.g., http://localhost:3001).
